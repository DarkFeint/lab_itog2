#Системные требования

Операционная система Windows 10 не ниже версии 1703

#Процесс запуска приложения

Зайти в папку "installer" и запустить установщик "Strannaya_fignya.exe"
Пройти весь процесс установки и два раза нажать на иконку на раочем столе

#Процесс запуска для разработки

Ипользовать программу-компилятор CMake не ниже версии 3.15
Использовать IDE MS Visual Studio 2019
